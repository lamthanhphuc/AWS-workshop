# Nhật ký công việc

## Tổng quan

Trong suốt **12 tuần thực tập** tại First Cloud Journey (từ 08/09/2025 đến 09/12/2025), tôi đã hoàn thành chương trình học AWS và phát triển dự án **Serverless Student Management System**.

### Giai đoạn 1: Học tập AWS Fundamentals (Tuần 1-7)

Trong 7 tuần đầu, tôi tập trung học và thực hành các dịch vụ AWS cốt lõi:

| Tuần | Chủ đề | Kỹ năng đạt được |
|------|--------|------------------|
| 1 | AWS Fundamentals | Hiểu Cloud Computing, AWS Global Infrastructure |
| 2 | IAM & Security | Tạo Users, Groups, Roles, MFA, Budget Alerts |
| 3 | VPC Basics | Thiết kế Subnets, Route Tables, IGW, NAT Gateway |
| 4 | VPC Security | Cấu hình Security Groups, NACLs, VPC Peering |
| 5 | Hybrid Networking | Transit Gateway, Route 53 Resolver, Hybrid DNS |
| 6 | EC2 | Instance Types, AMI, EBS, Auto Scaling Groups |
| 7 | Storage | AWS Backup, Storage Gateway, S3 Static Website |

### Giai đoạn 2: Phát triển Proposal Backend (Tuần 8-12)

Trong 5 tuần cuối, tôi phát triển Backend APIs cho hệ thống quản lý sinh viên:

| Tuần | Công việc | Kết quả |
|------|-----------|---------|
| 8 | Phân tích & Thiết kế | Architecture diagram, API design, DynamoDB schema |
| 9 | Class & Student APIs | 5 APIs: CRUD Class, List Students |
| 10 | Assignment & Grade APIs | 5 APIs: CRUD Assignment, Update Grades |
| 11 | Post API & Testing | 1 API, Integration tests, API documentation |
| 12 | Deploy & Demo | Deploy AWS, Proposal document, Presentation |

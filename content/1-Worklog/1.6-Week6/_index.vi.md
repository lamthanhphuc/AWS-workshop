### Kết quả đạt được tuần 6:

* Nắm vững lý thuyết về các dịch vụ Compute bổ sung:
  * EC2 Auto Scaling: Launch configurations, Auto Scaling Groups
  * Amazon EFS: Shared file storage cho EC2
  * Amazon FSx: Managed file systems (Windows, Lustre, NetApp, OpenZFS)
  * Amazon Lightsail: Simplified compute service
  * AWS MGN: Application migration service

* Thực hành thành công AWS Backup:
  * Tạo Backup plan với schedule
  * Thực hiện on-demand backup
  * Restore EC2 từ backup
  * Hiểu retention policies

* Triển khai AWS Storage Gateway:
  * Tạo File Gateway trên EC2
  * Cấu hình File Shares kết nối với S3
  * Mount file shares trên on-premises

* Triển khai S3 Static Website:
  * Cấu hình S3 bucket cho static hosting
  * Configure public access policies
  * Test website accessibility

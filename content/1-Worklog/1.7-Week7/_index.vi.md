### Kết quả đạt được tuần 7:

* Nắm vững lý thuyết Amazon S3:
  * S3 Storage Classes: Standard, IA, One Zone-IA, Glacier, Deep Archive
  * S3 Access Points cho multi-tenant access
  * S3 Lifecycle policies để tối ưu chi phí
  * S3 Versioning để bảo vệ data

* Hiểu S3 Security và Access Control:
  * Bucket Policies và IAM Policies
  * ACLs (Access Control Lists)
  * S3 Block Public Access
  * Pre-signed URLs

* Thực hành thành công:
  * Cấu hình CloudFront distribution cho S3
  * Enable và sử dụng Bucket Versioning
  * Thiết lập Cross-Region Replication
  * Di chuyển objects giữa Storage Classes

* Hiểu về AWS Snow Family:
  * Use cases cho data migration offline
  * Snowcone, Snowball Edge, Snowmobile

* Nắm vững Storage Gateway:
  * File Gateway, Volume Gateway, Tape Gateway
  * Hybrid cloud storage scenarios

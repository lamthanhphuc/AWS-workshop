### Kết quả đạt được tuần 2:

* Hiểu được kiến trúc Amazon VPC và các thành phần:
  * VPC, Subnets (Public/Private)
  * Route Tables và Internet Gateway
  * NAT Gateway cho private subnets
  * Elastic IP và Elastic Network Interface

* Nắm vững các khái niệm bảo mật mạng:
  * Security Groups (stateful firewall)
  * Network ACLs (stateless firewall)
  * VPC Flow Logs để giám sát traffic

* Hiểu các phương thức kết nối:
  * VPC Peering và Transit Gateway
  * VPN và Direct Connect
  * Các loại Load Balancer (ALB, NLB, CLB, GLB)

* Thực hành thành công:
  * Tạo VPC với public và private subnets
  * Cấu hình Route Tables
  * Thiết lập Internet Gateway và NAT Gateway
  * Tạo Security Groups và Network ACLs
